# Machine Learning Pipeline & Feature Engineering

This repository demonstrates the implementation of Feature Engineering and an End-to-End Machine Learning Pipeline using Python and Scikit-learn. It focuses on building reusable, modular, and production-ready machine learning workflows.


# Project Overview

The repository contains two Jupyter notebooks:

Feature Engineering
Data preprocessing
Feature scaling
Train-test splitting
Building preprocessing pipelines
Machine Learning Pipeline
Data loading
Data cleaning
Feature engineering
Model training
Model evaluation
Model serialization using Joblib

The project follows machine learning best practices by separating preprocessing and model training into reusable components.
